# 📚 College Notes Mongo

A simple and interactive **college notes sharing app** built using **Streamlit** and **MongoDB**. 
This project allows students to view, download and admin can upload notes directly in the browser.

## 🌐 Live Demo

👉 https://ecom-backend-py.onrender.com

## 📌 Features

- 📥 Download notes using direct PDF/Google Drive links
- 📤 Admin can upload new notes with subject & semester
- 🔐 Admin Login system for security
- 🗑️ Delete notes - only for admin
- 🔍 Search notes by subject name
- 🎯 Filter notes by semester
- 🎨 Clean and simple UI with Streamlit
- ⚡ Fast and lightweight

---

## 🛠️ Tech Stack

- **Frontend:** Streamlit
- **Backend:** Python
- **Database:** MongoDB Atlas
- **Hosting:** Render
